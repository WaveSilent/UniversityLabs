**Обьяснение:** Создаю класс Fraction, переменные которые будут использоваться в нем это int numerator - числитель и int denominator - знаменатель, добавляю свойства и для denominatora прописываю условие, что человек введет 0, то ему на экран вернется сообщение, что на 0 делить нельзя, и человек должен ввести новое значение для denominator, далее я создаю конструктор, в нем 2 параметра, числитель и знаменатель. Далее идет метод Sum(string fraction), параметром в нем является другая дробь, которую мы будем создавать в главном коде, она имеет тип переменной string, так как я буду передавать в параметр текстовое представление, пройдемся по функционалу данного метода: я создаю в нем переменные, которые мне помогут определить числитель, знаменатель, число ли это или дробь, в первую очередь я определяю циклом, если у другой string fraction "дробь = /" в строке, если нет, то я понимаю, что это число, преобразую стркоу в число, и после условиями проверяю, если это число = 0, и числитель с которым я складываю это число **не равно** 0, то я возвращаю числитель / знаменатель, иначе если число = 0, и числитель с которым я складываю это число **равно** 0, то вернется "0", иначе я к числителю прибавлю (число * на знаменатель), и верну текстовое представление.
